# **App Name**: Muscle Factory Gym

## Core Features:

- Comprehensive Content Sections: Implement all required content areas including Hero, Trust/Stats, About, Membership Plans, Services, Trainers, Gallery, Reviews, Contact, and Footer with a premium, clean UI.
- Mobile-First Responsive Design: Ensure the website is fully optimized for all devices, prioritizing mobile experience, including a sticky navbar and smooth scrolling functionality.
- Direct Communication Integrations: Integrate click-to-WhatsApp (https://wa.me/919028209504) and click-to-call (tel:9028209504) functionality for all relevant CTAs and contact points.
- Interactive Location Map: Embed an interactive Google Maps widget displaying the gym's location in Ulhasnagar for easy navigation and discovery.
- Dynamic Gym Image Gallery: Showcase gym interiors, equipment, and members through an engaging grid-based image gallery to visually attract potential members.
- Trust-Building Displays: Prominently feature elements such as 'Happy Members' count, average rating, 'Certified Trainers' highlights, and genuine member testimonials.
- Clear Membership Plan Presentation: Display various membership plans (Basic, Standard, Premium) with price placeholders and distinct CTA buttons for enrollment.

## Style Guidelines:

- Primary action color: A bold and dynamic red (#DB1932) to signify energy and professionalism, serving as a key accent.
- Background color: A deep, rich charcoal with a subtle hint of red (#1D1819), providing a premium dark theme foundation.
- Secondary accent color: A vibrant, clear orange (#FA6C06) used for additional highlights and call-to-action elements, offering strong contrast.
- Headlines: 'Space Grotesk' (sans-serif) for a modern, strong, and impactful presence that reflects a tech-forward fitness brand.
- Body text: 'Inter' (sans-serif) for exceptional readability, a clean, modern aesthetic, and neutrality across various content sections.
- Use minimalist and sharp fitness-themed icons that align with the professional and modern brand image, avoiding overly decorative elements.
- Adopt a clean, structured grid-based layout for content sections, ensuring visual hierarchy and ample negative space for a premium feel. All components are designed for a mobile-first approach.
- Implement subtle scroll-triggered animations and smooth transitions for section reveals and button interactions, enhancing the user experience without being distracting.
